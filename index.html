Operation Exodus

A collaborative team project developed with fellow students for the DES310 module. Created in the Unity Engine, Operation Exodus is a 2D top-down, rogue-like shooter Set on an infected refugee spaceship. My main contributions for this project were programming the enemy AI behaviour, tutorial, quality of life features and implementing the visual effects(animation, particles, lighting).
